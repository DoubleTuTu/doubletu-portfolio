# Double兔的个人作品集

七龙珠主题的个人编程作品展示网页

## 项目简介

这是一个用 Next.js 15 + TypeScript + Tailwind CSS 构建的个人作品集网站，采用七龙珠动漫主题设计。

### V1.0 功能

- ✅ 访问统计（Cookie 去重计数）
- ✅ 背景音乐播放器（循环播放、音量控制）
- ✅ 5 个项目展示（支持角色图片上传）
- ✅ 文章列表与详情
- ✅ AI 机器人助手（多轮对话）
- ✅ 管理后台（项目管理、文章上传）

## 技术栈

- **前端框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS + CSS-in-JS
- **数据存储**: 本地 JSON 文件
- **部署**: Vercel

## 快速开始

```bash
# 进入应用目录
cd app

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

访问 [http://localhost:3000](http://localhost:3000) 查看网站。

## 项目结构

```
app/
├── app/                    # Next.js 应用目录
│   ├── components/         # 组件
│   ├── lib/               # 工具函数
│   ├── api/               # API 路由
│   ├── data/              # 数据文件
│   ├── public/            # 静态资源
│   └── pages/             # 页面
├── docs/                   # 文档
└── README.md
```
