### 展示内容
- 项目名称
- 一句话简介

## 数据模型

### 项目（Projects）
- id
- name（项目名称）
- description（一句话简介）
- image_url（截图URL）
- link_url（网页链接）
- character（角色：goku/vegeta/piccolo/trunks/bulma）
- catchphrase（口头禅）
- sort_order（排序）

### 文章（Articles）
- id
- title（标题）
- content（内容）
- file_type（文件类型：word/pdf/markdown）
- published_at（发布时间）
- view_count（阅读量）
- created_at
- updated_at

### 留言（Messages）
- id
- nickname（昵称）
- content（内容）
- reply（回复）
- created_at

### 访问统计（Analytics）
- id
- total_views（总访问量）
- today_views（今日访问量）
- date（日期）

### 管理员（Admin）
- id
- username
- password
