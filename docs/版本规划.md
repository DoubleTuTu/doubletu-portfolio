# 版本规划 - 七龙珠主题个人作品集

## 产品概述

**核心定位**：七龙珠主题的个人编程作品展示网页

**核心价值**：用趣味性的动漫主题 + AI 机器人，让作品集脱颖而出，展示你的开发能力

**目标用户**：访客、潜在雇主/合作伙伴

---

## 核心需求清单

### 必须有（MVP）
- [x] 展示 5 个项目，可跳转到应用链接
- [x] 文章列表
- [x] AI 机器人

### 重要功能
- [ ] 角色交互（口头禅、动画）
- [ ] 后台管理系统
- [ ] 留言板

### 锦上添花
- [ ] 背景音乐
- [ ] 访问统计
- [ ] 项目截图
- [ ] 响应式设计

---

## 版本规划路线图

### V0.1 MVP - 能展示、能跳转

**目标**：最快速度上线一个能用的网站，让访客看到你的项目

**功能清单**：
1. 首页框架
   - 七龙珠主题颜色（橙色 + 蓝色 + 黄色）
   - 导航栏（首页、文章、建议）
   - 个人简介区域
2. 5 个项目卡片
   - 项目名称
   - 一句话简介
   - **可点击跳转到应用链接** ⭐ 核心
   - 七龙珠角色图标
3. AI 机器人（基础版）
   - 孙悟空头像
   - 右下角浮窗
   - 基础问答能力
4. 响应式设计（基础）

**不做什么**：
- 文章功能（暂不放"文章"链接）
- 留言板（暂不放"建议"链接）
- 角色口头禅、动画
- 后台管理
- 背景音乐
- 访问统计
- 项目截图

**验证点**：你能把这个网站发给朋友看，朋友能点击项目卡片跳转到你的应用

**预计工作量**：3-5 天

---

### V0.2 - 增加文章功能

**目标**：可以展示你的技术文章

**功能清单**：
1. 文章列表页
   - 文章标题
   - 发布时间
   - 阅读量
   - 阅读全文链接
2. 文章详情页
   - 文章内容展示（支持 Markdown）
3. 后台上传文章（简单版）
   - 上传 Markdown 文件
   - 填写标题
   - 删除文章

**验证点**：你能上传并展示一篇文章

**预计工作量**：2-3 天

---

### V0.3 - 增加互动效果

**目标**：让网站更有趣、更有动漫感

**功能清单**：
1. 角色口头禅
   - 点击卡片触发气泡显示
   - 每个角色的技能口号
2. 鼠标悬停动画
   - 角色出现动画
   - 卡片高亮效果
3. AI 机器人完善
   - 知识库接入（基于上传的文章）
   - 更好的问答体验

**验证点**：朋友觉得这个网站很有趣，口头禅很好玩

**预计工作量**：2-3 天

---

### V0.4 - 后台管理系统

**目标**：你可以自己管理内容

**功能清单**：
1. 管理员登录
   - 账号密码登录
   - 简单的 session 管理
2. 文章管理
   - 上传文章（Word、PDF、Markdown）
   - 编辑文章标题
   - 删除文章
   - 修改阅读量
3. 项目管理
   - 增加项目
   - 删除项目
   - 修改项目信息（名称、简介、链接、截图）
   - 更换角色
   - 修改口头禅
   - 调整卡片顺序
4. 留言管理
   - 查看留言
   - 回复留言
   - 删除留言

**验证点**：你能自己添加/修改项目和文章，不需要改代码

**预计工作量**：3-5 天

---

### V0.5 - 留言板功能

**目标**：让访客可以给你留言

**功能清单**：
1. 留言板页面
   - 留言表单（昵称、内容）
   - 留言列表显示
   - 管理员回复显示
2. 后台留言管理
   - 删除不当留言

**验证点**：朋友能在你的网站留言

**预计工作量**：1-2 天

---

### V1.0 - 完整版

**目标**：功能完整、体验良好、可以自豪地分享

**功能清单**：
1. 背景音乐系统
   - 3 首歌曲循环播放
   - 自动播放（首次访问）
   - 显示当前歌曲名称
   - 暂停按钮
   - 音量控制
2. 访问统计
   - 总访问量
   - 今日访问量
   - 实时更新
   - 后台查看
3. 项目截图
   - 每个项目卡片显示 4:3 截图
   - 后台上传截图
4. 响应式设计完善
   - 手机端优化
   - 卡片竖排显示
5. UI/UX 打磨
   - 动画效果优化
   - 色彩调整
   - 细节完善
6. 文章格式支持完善
   - Word、PDF 解析

**验证点**：网站功能完整，体验流畅，可以对外分享

**预计工作量**：2-3 天

## 技术栈确认

| 语言 | TypeScript |
| 样式 | Tailwind CSS |

| 部署 | Vercel |

---

## 待确认的关键信息

1. **5 个项目的实际链接**：需要你提供 5 个项目的在线地址
2. **项目的截图**：V1.0 时需要，暂时不需要
3. **文章内容**：你有现成的文章吗？格式是什么？
4. **七龙珠角色图片**：使用官方图片还是 AI 生成？
5. **背景音乐文件**：你需要自己准备 3 首 MP3 文件

## 下一步行动

### 立即开始 V0.1

1. **初始化 Next.js 项目**
   ```bash
   npx create-next-app@latest portfolio --typescript --tailwind
   ```

2. **准备项目数据**
   - 5 个项目的名称、简介、链接
   - 可以先用假数据，后面再替换

3. **设计首页布局**
   - 头部（头像 + 名字 + 导航）
   - 个人简介
   - 项目卡片网格
   - AI 机器人浮窗

4. **实现项目卡片点击跳转**
   - 确保 5 个项目都能跳转到正确链接

5. **基础 AI 机器人**
   - 先用简单版本，能问答即可
